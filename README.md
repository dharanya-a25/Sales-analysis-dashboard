Project Description: A retail company wants to analyze its sales performance across products,
regions and salespersons. The dashboard should help management understand sales trends and
identify top performing areas.
Dataset Fields
- Date
- Product
- Category
- Region
- Salesperson
- CustomerSegment
- Quantity
- SalesAmount
- Profit
- Target
Dashboard Tasks
- Display total sales, profit and quantity using KPI card visuals.
- Show the trend of sales over time using a line chart.
- Analyze sales by region using a column chart.
- Identify top selling products using a bar chart.
- Show category contribution to total sales using a pie or donut chart.
- Compare salesperson performance using a column chart.
- Compare sales and profit using a scatter chart.
- Add slicers for Date, Region, Category and Customer Segment.
